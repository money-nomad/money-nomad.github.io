# GEMINI.md - Money Nomad Landing Page

## Directory Overview
This directory contains the source code for the official landing page of **Money Nomad**, a privacy-focused, map-centric personal finance tracker designed for travelers and digital nomads. The project is hosted on GitHub Pages (`money-nomad.github.io`).

The site is a simple, static web application consisting of a landing page and a privacy policy. It serves as the public face of the Money Nomad mobile application (available on Android and iOS).

## Key Files
- **`index.html`**: The main landing page. It showcases the app's core value propositions (Map Tracking, Smart Tags, Multi-Currency, Offline-First) and provides links to the community discussions and bug reporting on GitHub.
- **`privacy.html`**: The privacy policy for the Money Nomad application, emphasizing its "Local-First" data storage philosophy and lack of user tracking.
- **`assets/`**: Contains visual assets used on the website:
    - `logo.avif`: The Money Nomad branding logo.
    - `overview-sales-pitch.avif`: Hero image for the landing page.
    - `transaction_view.avif`, `maps_view.avif`, `stats_view.avif`, `settings_view.avif`: Screenshots of the mobile application in action.

## Usage
- **Content Updates**: To update the landing page or privacy policy, edit `index.html` or `privacy.html` directly.
- **Assets**: New screenshots or branding materials should be placed in the `assets/` directory and referenced in the HTML files.
- **Deployment**: Changes pushed to the main branch are automatically deployed via GitHub Pages to [money-nomad.github.io](https://money-nomad.github.io).

## Development Philosophy
- **Privacy-First**: The landing page reflects the app's core values—no analytics, no tracking, and local-first data.
- **Simplicity**: The site uses vanilla HTML and CSS (internal styles) to maintain a lightweight and fast-loading presence.
